  • Firebase Console kholo: console.firebase.google.com

  • Google Analytics (optional) - Enable/Disable karo

üîë Authentication Enable Karo

Users ko login karne ke liye authentication setup karna padega:

  • Left sidebar se Authentication pe click karo
  • Click "Get Started"
  • Sign-in method tab open karo
  • Email/Password enable karo (toggle ON)
  • Google enable karo (toggle ON)
  • Save karo
‚ö†Ô∏è Important: Dono methods enable karna zaruri hai - Email/Password aur Google sign-in.

üíæ Firestore Database Banao

Data store karne ke liye database chahiye:

  • Left sidebar se Firestore Database click karo
  • Click "Create Database"
  • Production mode select karo
  • Location: asia-south1 (India ke liye best)
  • Click "Enable"

Database ban gaya! Ab security rules set karo:

// Rules tab mein ye paste karo:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read: if request.auth != null;
      allow write: if request.auth.uid == userId;
    }
  }
}

Click "Publish" to save rules.

üëë User Ko Admin Banao

Ye sabse important step hai - User ko admin rights dena:

  • Firebase Console wapas kholo
  • Firestore Database pe jao
  • users collection kholo
  • Apna user document dhundho (email se identify karo)
  • Document pe click karo to edit karo
  • "role" field mein "user" ko "admin" mein change karo
  • ‚úÖ Save karo
‚ö†Ô∏è Important: "admin" lowercase mein likhna hai (not "Admin" or "ADMIN"). Case-sensitive hai!

Document aisa dikhega:

users (collection)

üìä Admin Dashboard Features
